# entp.jp

ENTP向けの診断・ゲーム・ネタコンテンツをまとめたサイトです。  
議論好き、ひらめき型、ツッコミ気質な人が楽しめるWebコンテンツを制作しています。

## About

entp.jp is a Japanese web project focused on MBTI-style personality content, especially for ENTP users.

## Concept

「正しいか」だけじゃなく、  
「面白いか」「ひねりがあるか」「遊びたくなるか」を大事にしたサイトです。

## Features

- ENTP向け診断コンテンツ
- MBTI系のゲーム・企画
- ネタ寄りのタイプ解説
- スマホ対応のシンプルなWebサイト
- SNSで共有しやすい結果ページ

## Tech Stack

- HTML
- CSS
- JavaScript
- Firebase / Vercel など必要に応じて
